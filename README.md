# 🐍 Learning Python

A structured and disciplined roadmap to mastering Python—from fundamentals to advanced applications like data science, machine learning, and system-level problem solving.

---

## 📌 Objective

* Build strong Python fundamentals
* Develop problem-solving ability (DSA)
* Gain practical skills through projects
* Prepare for real-world applications (ML, automation, backend)

---

## 🧠 Roadmap

### 🔰 Phase 1: Core Python

* Syntax, Output, Comments
* Variables, Data Types
* Strings, Booleans, Operators
* Lists, Tuples, Sets, Dictionaries
* If...Else, Match
* Loops (for, while)
* Functions, Range
* Input Handling
* Exception Handling

---

### 🧱 Phase 2: Intermediate Python

* File Handling (read/write/delete)
* Modules & Packages
* JSON, RegEx
* Virtual Environments (venv)
* Iterators & Generators

---

### 🧬 Phase 3: Object-Oriented Programming

* Classes & Objects
* **init** and self
* Class Methods & Properties
* Inheritance
* Polymorphism
* Encapsulation

---

### 🧮 Phase 4: Data Structures & Algorithms

* Arrays, Lists
* Stacks, Queues
* Linked Lists
* Hash Tables
* Trees (BST, AVL)
* Graphs
* Searching (Linear, Binary)
* Sorting (Bubble → Merge → Quick)

---

### 📊 Phase 5: Libraries & Data Handling

* NumPy
* Pandas
* SciPy (basic)

---

### 📈 Phase 6: Data Visualization

* Matplotlib (plots, histograms, scatter, etc.)

---

### 🤖 Phase 7: Machine Learning (Basics)

* Statistics (mean, std, distributions)
* Regression (linear, polynomial)
* Train/Test split
* Decision Trees
* Clustering (K-means)
* Logistic Regression

---

### 🗄 Phase 8: Databases

#### MySQL

* CRUD operations
* Joins, filtering, sorting

#### MongoDB

* Collections
* CRUD operations

---

### ⚡ Phase 9: Practical Skills

* Reverse a string
* Remove duplicates
* Basic algorithm problems
* Small automation scripts

---

## 🚀 Execution Strategy

* Focusing on one phase at a time
* Building at least 1 project per phase
* Revising consistently
* Avoiding tutorial overload
